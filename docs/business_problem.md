# Business Problem Statement

Financial institutions face significant challenges in identifying high-risk borrowers while maintaining profitability. Poor risk assessment can result in loan defaults, increased exposure, and reduced interest income.

This project aims to address the following business problems:

1. How can borrowers be segmented based on credit risk?
2. Which financial indicators most strongly influence loan risk?
3. What customer profiles contribute to high exposure?
4. How does loan structure impact long-term risk?
5. How can decision-makers monitor risk trends effectively?

By analyzing historical loan data, the project provides insights that support smarter credit approval, pricing strategies, and risk control.
